# Введение

Добро пожаловать в документацию Reqst. Reqst — это non-custodial процессинг криптоплатежей, который позволяет принимать оплату напрямую на ваш кошелек.

## Основные возможности

- **Direct-to-Wallet**: Мы никогда не удерживаем ваши средства.
- **Мультичейн**: Поддержка TON, TRON, Solana и EVM.
- **0% комиссии**: Никаких комиссий с оборота, только фиксированная подписка.

## С чего начать

Для начала работы создайте аккаунт в [Консоли мерчанта](/app/auth) и сгенерируйте ваш первый API ключ.
