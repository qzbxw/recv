# Introduction

Welcome to the Reqst documentation. Reqst is a non-custodial crypto payment processor that allows you to accept payments directly to your wallet.

## Core Features

- **Direct-to-Wallet**: We never hold your funds.
- **Multi-chain**: Support for TON, TRON, Solana, and EVM.
- **Zero Fees**: No turnover commissions, just a flat subscription.

## Getting Started

To get started, create an account in our [Merchant Console](/app/auth) and generate your first API key.
