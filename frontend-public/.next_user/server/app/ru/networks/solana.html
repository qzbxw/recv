SOLANA NETWORK

Solana-платежи для быстрого developer-owned checkout.

recv добавляет SOL и поддерживаемые SPL assets в единый payment API, чтобы web3-продукты создавали инвойсы и реагировали на подтвержденные переводы без собственной watcher logic.

SUPPORTED ASSETS

Платежи в USDT SPL на Solana.

Solana подходит командам, которым нужны быстрые платежи в стейблкоинах в сети Solana.

01

SPL USDT

Стандартный стейблкоин в сети Solana для быстрых и дешевых долларовых платежей.

WHY SOLANA

Быстрый payment feedback для web3 users.

Solana сильна там, где покупатели уже используют Solana wallets и ожидают быстрое подтверждение в developer tools, apps или digital goods flows.

LIMITATIONS AND NOTES

Asset и wallet selection должны быть точными.

Покупатель должен отправить настроенный Solana asset в сети Solana. Token account behavior, поддержка кошельков и временная congestion могут влиять на платежный опыт.

CHECKOUT AND API MECHANICS

Единый invoice flow для SOL и SPL assets.

01

Создание инвойса

Backend создает Solana invoice через recv API или использует hosted Checkout.

02

Показ wallet data

Checkout показывает сеть, актив, сумму, адрес и QR в buyer-friendly экране.

03

Watch transfer

recv отслеживает native или SPL transfers и сопоставляет оплату с invoice.

04

Trigger backend

Signed webhook сообщает продукту, когда нужно открыть доступ, выдать товар или обновить статус.

RELEVANT USE CASES

Лучше всего для web3-native продуктов.

Solana полезна, когда платежи являются частью продукта для разработчиков, creators или crypto-native покупателей.

01

Developer tools

Создавайте инвойсы из backend и используйте webhooks для automated access.

02

Цифровые товары

Продавайте ключи, downloads или лицензии Solana wallet users.

03

Web3 apps

Добавьте быстрый payment option для пользователей Solana ecosystem.
